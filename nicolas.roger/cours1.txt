commande ping :

ping allows you to ping a remote machine, generally to check if the machine is
still alive.

ping on google : 

ping 8.8.8.8

if the command works, you'll have some informations like the time of response.
With it you can know if there is some latency for example.


if you want to ping with a defined interval, you can use the flag -i : 

example : 

ping -i 5 8.8.8.8

ping google every 5s.
